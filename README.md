# AI Chatbot 🤖

A simple AI-based chatbot built using Python.
Created by Maneesha | B.Tech AIML | Bahra University

## About This Project
This chatbot uses Natural Language Processing (NLP) 
concepts to understand user input and give smart replies.

## Features
- Answers questions about Artificial Intelligence
- Answers questions about Machine Learning
- Tells programming jokes 😄
- Built using Python 3

## Technologies Used
- Python 3
- NLP (keyword matching technique)

## How to Run
1. Install Python on your computer
2. Download chatbot.py
3. Open terminal and type:
   python chatbot.py
4. Start chatting!

## Sample Conversation
You: what is machine learning
ManeeBot: ML is a subset of AI where machines learn from data!

You: joke
ManeeBot: Why do programmers prefer dark mode? Light attracts bugs!
